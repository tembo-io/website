Configuration and Management

Tembo Cloud offers you a wide range of tools to help manage your deployment. Find out more in these how-to guides, or contact us at support@tembo.io.

📄️ Backup and restore

Tembo performs Postgres backups and Write-Ahead Log (WAL) archiving using Barman, powered by the Cloud Native Postgres Operator (CNPG).

📄️ Connection Pooling

A connection pooler is a tool used to manage database connections, sitting between your application and Postgres

📄️ High Availability

Tembo supports highly available (HA) deployments of Postgres in Tembo Cloud. Click the HA toggle on when creating or updating an instance to enable this feature. When using the API directly, configure replicas to 2.

Configure custom domains that are in your Postgres connection URLs.
